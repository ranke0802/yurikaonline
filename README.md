# 🌸 유리카 온라인 (Yurica Online) v0.7 (Ergonomic & Performance)

### 🔄 최근 업데이트 내역 (v0.7)
- **컨트롤러 최최종 최적화**: 조이스틱과 공격 버튼을 **화면 가장자리**(`10~15px`)로 바짝 붙이고 크기를 한 번 더 키워 엄지손가락 조작 범위를 최적화했습니다.
- **아이템 드랍 최적화 (랙 해결)**: 아이템 드랍 시 발생하던 프레임 드랍(랙)을 해결하기 위해 고비용의 실시간 그래디언트 연산을 단순화된 그래픽으로 대체했습니다.
- **UI 레이아웃 안정화**: 퀘스트 창과 상태 로그 간의 간격과 크기를 재조정하여 서로 가려지지 않도록 쌓기 로직을 강화했습니다.
- **기능 복구**: 이제 게임 중에도 자유롭게 **화면 캡쳐**를 하실 수 있습니다.

# 🌸 유리카 온라인 (Yurica Online) v0.6

### 🔄 최근 업데이트 내역 (v0.6)
- **컨트롤러 대폭 확대**: 조이스틱과 공격/스킬 버튼의 크기를 기존 대비 **1.5배 키우고 위치를 상향 조정**하여 조작 편의성을 극대화했습니다.
- **UI 겹침 완전 해결**: 퀘스트 창과 채팅창(상태 로그)을 하나의 컨테이너로 묶어 세로로 자동 스택되게 수정했습니다. 이제 두 창이 서로 가려지지 않습니다.
- **화면 캡쳐 및 시스템 메뉴 복구**: 보안상 막혀있던 시스템 호출 옵션을 완화하여 **화면 캡쳐 및 브라우저 시스템 메뉴**를 정상적으로 사용할 수 있습니다.
- **디자인 세밀 조정**: 조이스틱 위치와 액션 버튼 간의 간격을 Galaxy S23 등 최신 기기의 베젤 및 시스템 바 위치에 맞춰 재조정했습니다.

# 🌸 유리카 온라인 (Yurica Online) v0.5

### 🔄 최근 업데이트 내역 (v0.5)
- **UI 완전 재배치**: 가방, 내 정보, 전체화면 버튼을 미니맵 하단으로 통합하여 우측 상단으로 모았습니다.
- **채팅창 최적화**: 채팅창을 좌측 퀘스트 리스트 하단으로 이동시켜 가독성을 높이고 조이스틱과의 충돌을 방지했습니다.
- **선택 및 팝업 방지**: 버튼 클릭 시 이모지가 텍스트로 선택되거나 복사 탭이 뜨는 현상을 완벽 차단했습니다.
- **모바일 반응 속도 개선**: 터치 입력(`touchstart`) 최적화를 통해 S23 등 모바일 기기에서의 버튼 반응 속도를 대폭 향상했습니다.
- **레이아웃 안정화**: 버튼 눌림(Active) 효과 추가 및 전체적인 UI 좌표를 재조정했습니다.

# 🌸 유리카 온라인 (Yurica Online) v0.2

환상적인 픽셀 아트 월드에서 꿈을 찾아 떠나는 웹 기반 MMORPG 프로젝트, **유리카 온라인**의 첫 번째 공개 버전입니다.

## 🌟 주요 특징
- **감성적인 픽셀 그래픽**: AI와 협업하여 제작한 정교한 푸른 마법사 캐릭터와 화사한 꽃밭 맵.
- **부드러운 조작감**: 키보드(방향키) 및 마우스 클릭을 통한 자유로운 8방향 이동 시스템.
- **실시간 크로마키 렌더링**: 캐릭터 스프라이트의 배경을 실시간으로 제거하여 맵과 완벽하게 조화되는 그래픽 구현.
- **RPG 핵심 UI**: 퀘스트 리스트, 미니 채팅창, 체력/마나 바, 스킬 슬롯 등 게임 몰입감을 높이는 사용자 인터페이스.

## 🎮 플레이 방법 (PC)
- **이동**: 방향키(Arrow Keys) 또는 WASD
- **달리기**: 한 방향으로 1초 이상 이동 (속도 1.3배 가속)
- **기본 공격**: J (레이저)
- **스킬**: H (매직 미사일), U (파이어볼), K (보호막)
- **메뉴**: Shift+B (가방), Shift+I (내 정보)

## 📱 플레이 방법 (Mobile)
- **이동**: 가상 조이스틱
- **전투**: 스킬 버튼 터치
- **전체화면**: 하단 📺 버튼 클릭

## 🔄 최근 업데이트 내역 (v0.2)
### 1. 전투 시스템 및 스킬 리워크
- **기본 공격**: 관통형 레이저 공격으로 변경 (마나 1 회복)
- **매직 미사일**: 호밍 유도 기능 (마나 2 소모)
- **파이어볼**: 직선 범위 공격 및 화상 도트 데미지 추가 (마나 5 소모)
- **보호막**: 3분간 피해 80% 감소 및 마나 우선 소모 (마나 20 소모)

### 2. 고도화된 몬스터 AI 및 전리품
- **추격 AI**: 몬스터가 일정 거리 내의 플레이어를 감지하고 추격
- **공격 AI**: 플레이어 근접 시 주기적인 데미지 발생
- **회복 시스템**: 몬스터 처치 시 확률적으로 HP 회복 구슬(+) 드랍
- **무한 스폰**: 맵 전체 영역에 대해 3초마다 몬스터 자동 생성

### 3. 기동성 강화: 달리기 시스템
- **가속**: 1초 이상 연속 이동 시 '달리기' 상태 진입 (속도 1.3배)
- **유예 시간**: 방향 전환 시 0.5초의 유예 시간을 주어 달리기 상태가 끊기지 않도록 개선
- **이펙트**: 달리기 시 발자국 먼지 파티클 효과 추가

### 4. 편의성 및 모바일 최적화
- **미니맵 추가**: 우측 상단에 플레이어(백색) 및 몬스터(적색) 위치 실시간 표시
- **S23 최적화**: 모바일 화면 크기에 맞게 UI 규모를 1.5배 축소하고 엄지손가락 영역에 최적화된 배치
- **전체화면 지원**: 브라우저 UI 없이 플레이 가능한 Fullscreen API 연동

## 🛠 기술 스택
- **Language**: JavaScript (Vanilla JS ES6+)
- **Graphics**: HTML5 Canvas API
- **Styling**: Vanilla CSS
- **Assets**: Midjourney / DALL-E 기반 픽셀 아트 소스
