[**🎮 Firebase 라이브 데모**](https://yurika-online.web.app/)

## 📜 최신 업데이트 (v0.23.0) - 2026-01-25
- **체인 라이트닝 개편**: 이제 기본 공격(J) 시 마나를 소모하지 않습니다.
- **마나 흡수 시스템**: 적중 시 적의 체내 전기를 과부하 시켜 연쇄 감전을 일으키고 마나를 일부 흡수합니다.
- **밸런스 조정**: 슬라임의 이동 속도를 초당 30px로 하향 조정하여 게임 초반 난이도를 조절했습니다.
- **문구 업데이트**: 스킬창의 체인 라이트닝 설명을 세계관에 맞춰 보강했습니다.

## �️ 조작 방법
- **이동**: 화면 좌측 조이스틱 또는 방향키
- **기본 공격 (J)**: 체인 라이트닝 (연쇄 번개)
- **스킬 (H)**: 매직 미사일 (유도탄)
- **스킬 (U)**: 파이어볼 (광역 폭발)
- **스킬 (K)**: 앱솔루트 베리어 (방어막)
- **메뉴**: 화면 우측 상단 아이콘 (가방, 스킬, 내 정보)

## �️ 기술 스택
- **Frontend**: HTML5 Canvas, Vanilla JavaScript, CSS3
- **Backend**: Firebase (Authentication, Realtime Database, Hosting)
- **PWA**: Service Worker 및 Manifest 지원

## � 주요 기능
- **실시간 멀티플레이어**: 다른 플레이어와 같은 필드에서 모험을 즐길 수 있습니다.
- **성장 시스템**: 몬스터를 처치하고 경험치를 얻어 레벨업하고 스탯/스킬을 강화하세요.
- **퀘스트 시스템**: 다양한 퀘스트를 수행하고 보상을 획득하세요.
- **자동 저장**: 플레이어의 데이터(레벨, 스탯, 골드 등)가 실시간으로 안전하게 저장됩니다.
