# 🌸 유리카 온라인 (Yurika Online) v1.23 (Mobile UX & Global Scaling)
[**🎮 지금 바로 플레이하기 (Live Demo)**](https://ranke0802.github.io/yurikaonline/)

### 🔄 최근 업데이트 내역 (v1.23)
- **전역 화면 배율(Zoom) 기능 도입**: 
    *   게임 화면을 전체적으로 약 **70% 수준으로 줌 아웃**하여, 캐릭터와 몬스터는 작게 보이면서도 맵의 더 넓은 범위를 한눈에 볼 수 있게 개선했습니다.
- **모바일 UX 및 가로 모드(Landscape) 최적화**:
    *   **가로 모드 UI 전면 개편**: 가로 모드에서도 퀘스트와 채팅창을 확인할 수 있도록 배치하고, 미니맵 비율을 조정했습니다.
    *   **조작 편의성 강화**: 모바일에서 조이스틱과 스킬 버튼이 화면 구석에 너무 붙어 있어 조작이 불편하던 문제를 해결하기 위해 버튼 위치를 안쪽으로 조정했습니다.
    *   **UI 투명도 적용**: 배경의 몬스터 시인성을 확보하기 위해 모바일의 모든 주요 UI 패널에 **반투명(rgba 0.6)** 및 블러 효과를 적용했습니다.
- **미니맵 디자인 개선**: 미니맵 배경과 메뉴에 투명도를 적용하여 더욱 세련된 premium 룩을 구현했습니다.

### 🔄 최근 업데이트 내역 (v1.21)
- **UI/UX 프리미엄 리뉴얼 및 편의성 강화**:

        - 스탯 버튼이 사라지던 방식을 **비활성(회색) 상태 유지**로 변경하여 UI 레이아웃 흔들림을 방지했습니다.
    *   **업데이트 내역 스티키 헤더 도입**: 패치 노트를 길게 스크롤해도 닫기 버튼이 항상 우측 상단에 고정되도록 개선했습니다.
- **기능 안정성 및 시스템 최적화**:
    *   **전체화면 시스템 안정화**: PC 버전에서 버튼이 간헐적으로 작동안하던 문제와 모바일에서 중복 호출로 에러가 뜨던 현상을 완전히 해결했습니다.
    *   **시스템 로그 정밀화**: 불필요한 브라우저 레벨의 권한 에러 메시지를 숨기고 게임 내 시스템 메시지를 정돈했습니다.

### 🔄 최근 업데이트 내역 (v1.20)
- **신규 스탯 시스템 및 재생 로직 도입**:
    *   **신규 파생 스탯 추가**: 이제 **방어력(DEF)**, **체력 회복력**, **마나 회복력** 스탯이 추가되었습니다.
    *   **체력(VIT) 효율 개편**: VIT 1당 **최대 체력 +10**, **방어력 +1**, **체력 회복력 +1**이 증가합니다.
    *   **지혜(WIS) 효율 개편**: WIS 1당 **최대 마나 +10**, **마나 회복력 +1**이 증가하며, **지혜 2당 공격력 1**이 추가로 상승합니다.
    *   **지력(INT) 효율 개편**: INT 1당 **공격력 +1**이 정비례하여 상승합니다.
    *   **자동 회복(Regen) 시스템**: 전투 중이 아닐 때(2초 대기 시) 매 초마다 각 **회복력 스탯 수치만큼 HP와 MP가 자동으로 회복**됩니다.
    *   **전투 공식 반영**: 몬스터에게 받는 피해가 방어력 수치만큼 감소합니다. (최소 1 데미지)
    *   **UI 업데이트**: '내 정보' 창에서 방어력과 체력/마나 회복력을 실시간으로 확인할 수 있습니다.

- **이동**: 방향키(Arrow Keys) 또는 WASD
- **달리기**: 한 방향으로 1초 이상 이동 (속도 1.3배 가속)
- **기본 공격**: J (레이저)
- **스킬**: H (매직 미사일), U (파이어볼), K (보호막)
- **메뉴**: Shift+B (가방), Shift+I (내 정보)

## 📱 플레이 방법 (Mobile)
- **이동**: 가상 조이스틱
- **전투**: 스킬 버튼 터치
- **전체화면**: 하단 📺 버튼 클릭
