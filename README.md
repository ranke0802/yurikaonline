## 📜 최신 업데이트 (v0.29.6) - 2026-01-26
- **부활 동기화 완전 해결 (Real-Fix)**:
  - **비석 유지 문제 해결**: 캐릭터 부활 시 데이터베이스 저장 뿐만 아니라 실시간 HP 동기화 패킷을 즉시 전송하도록 수정하여, 상대방 화면에서도 즉각 생존 상태로 전환되도록 했습니다. (이제 진짜로 비석이 사라집니다!)

## 📜 이전 업데이트 (v0.29.5)
- **네트워크 이동 보간 고도화**:
  - **끊김 현상 제거**: 모바일 유저의 움직임이 PC에서 끊겨 보이던 현상을 해결하기 위해 패킷 전송 주기를 60ms로 단축했습니다.
  - **부드러운 주행(Smoothing)**: 다음 위치로 순간이동 하는 대신 부드럽게 따라가는 보간 알고리즘(Lerp)을 적용하여 시각적 박진감을 높였습니다.
  - **버퍼 최적화**: 네트워크 지연에 대비한 버퍼를 200ms로 확대하여 불안정한 환경에서도 부드러운 움직임을 보장합니다.

## 📜 이전 업데이트 (v0.26.3)
- **채팅 포커스 버그 수정**:
- **채팅 편의성 대폭 개선**:
  - **Enter 키 순환 로직**: `Enter` → 채팅창 활성화 → 입력 → `Enter` → 메시지 전송 및 **자동 포커스 해제**로 이어지는 자연스러운 플로우를 구현했습니다.
  - **포커스 자동 복원**: 채팅 입력을 마치면 즉시 게임 화면으로 포커스가 돌아가 끊김 없는 조작이 가능합니다.

## 📜 이전 업데이트 (v0.26.1)
- **스킬 메커니즘 개선**:
  - **이동 불가 시스템**: 체인 라이트닝(J) 및 매직 미사일(H) 사용 시 캐릭터가 제자리에 고정되어 스킬 애니메이션에 몰입감을 더했습니다.
  - **매직 미사일 연출 개선**: 스킬을 꾹 누르고 있을 때 마법진이 끊기지 않고 유지되도록 개선했습니다.
- **채팅 시스템 최적화**:
  - **ESC 지원**: 입력 중 `ESC` 키로 채팅창 포커스를 해제할 수 있습니다.
  - **한글 입력(IME) 수정**: 한글 입력 시 마지막 글자가 중복 전송되던 현상을 해결했습니다.
  - **정지 로직**: 채팅 활성화 시 이동 중이던 캐릭터가 즉시 정지합니다.

## 🛠️ 로컬 개발 환경 구축
1. **의존성 설치**: `npm install`
2. **실행**: `npm start` (브라우저에서 `localhost:8080` 접속)

## 🎮 조작 방법
- **이동**: 화면 좌측 조이스틱 또는 방향키
- **기본 공격 (J)**: 체인 라이트닝 (연쇄 번개)
- **스킬 (H)**: 매직 미사일 (유도 레이저)
- **스킬 (U)**: 파이어볼 (광역 폭발)
- **스킬 (K)**: 앱솔루트 베리어 (방어막)

## 🚀 주요 기능
- **실시간 멀티플레이어**: 다른 플레이어와 같은 필드에서 모험을 즐길 수 있습니다.
- **성장 시스템**: 몬스터를 처치하고 경험치를 얻어 레벨업하고 스탯/스킬을 강화하세요.
- **자동 저장**: 플레이어의 데이터가 Firebase를 통해 실시간으로 안전하게 저장됩니다.
