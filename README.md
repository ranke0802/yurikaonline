# 🌸 유리카 온라인 (Yurica Online) v1.0 (Advanced UI & Balance)

### 🔄 최근 업데이트 내역 (v1.0)
- **스탯 분배 및 관리 시스템 고도화**:
    - **수동 스탯 투자**: 레벨업 시 주어지는 3개의 포인트를 원하는 능력치에 투자할 수 있습니다.
    - **저장 확인 시스템**: 정보창을 닫을 때 변경된 스탯을 저장할지 묻는 **인게임 전용 모달**이 출력됩니다. 실수로 올린 스탯은 '취소'를 통해 되돌릴 수 있습니다.
    - **기초 스탯 조정**: 새로운 캐릭터의 시작 스탯이 변경되었습니다 (체력 1, 지능 5, 지혜 2, 순발력 1).
- **스킬 시스템 정밀화**:
    - **개별 쿨타임 및 오버레이**: 스킬별로 개별 쿨타임이 적용되며, 아이콘 위에 **남은 시간(초)**과 **시계 방향 회색 오버레이** 효과가 실시간으로 표시됩니다.
    - **단축키 배지**: 각 버튼 우측 상단에 단축키(J, H, U, K)가 표시되어 편리한 조작을 돕습니다.
    - **전방위 툴팁 시스템**: 메인 액션 버튼은 물론, **스킬 창(K) 내의 스킬 아이콘**에 마우스를 올리거나 터치하고 있으면 상세 설명이 출력됩니다.
- **전투 시스템 및 피드백 강화**:
    - **크리티컬 연출 극대화**: 치명타 발생 시 **"Critical"** 문구 출력과 함께 데미지 숫자가 **2.5배 크기**로 표시됩니다.
    - **실시간 수치 피드백**: 체력 회복(+초록), 마나 소비(-파란), 기본 공격 마나 회복(+파란) 수치가 캐릭터 위에 텍스트로 즉각 표시됩니다.
    - **체태 게이지 개선**: HP/MP 바 중앙에 `현재 / 최대` 수치가 텍스트로 직접 추가되어 정밀한 관리가 가능합니다.
- **버그 수정 및 모바일 최적화**:
    - **마나쉴드 100% 흡수**: 마나쉴드 활성화 시 **체력 소모가 0**이 되며, 모든 데미지를 마나로 흡수합니다. (마나가 부족할 경우에만 초과분이 체력으로 전달됩니다.)
    - **모바일 시스템 피드백**: 모바일 기기에서의 시스템 메시지 출력 창 크기를 확장하여 가독성을 높였습니다.

# 🌸 유리카 온라인 (Yurica Online) v0.2

환상적인 픽셀 아트 월드에서 꿈을 찾아 떠나는 웹 기반 MMORPG 프로젝트, **유리카 온라인**의 첫 번째 공개 버전입니다.

## 🌟 주요 특징
- **감성적인 픽셀 그래픽**: AI와 협업하여 제작한 정교한 푸른 마법사 캐릭터와 화사한 꽃밭 맵.
- **부드러운 조작감**: 키보드(방향키) 및 마우스 클릭을 통한 자유로운 8방향 이동 시스템.
- **실시간 크로마키 렌더링**: 캐릭터 스프라이트의 배경을 실시간으로 제거하여 맵과 완벽하게 조화되는 그래픽 구현.
- **RPG 핵심 UI**: 퀘스트 리스트, 미니 채팅창, 체력/마나 바, 스킬 슬롯 등 게임 몰입감을 높이는 사용자 인터페이스.

## 🎮 플레이 방법 (PC)
- **이동**: 방향키(Arrow Keys) 또는 WASD
- **달리기**: 한 방향으로 1초 이상 이동 (속도 1.3배 가속)
- **기본 공격**: J (레이저)
- **스킬**: H (매직 미사일), U (파이어볼), K (보호막)
- **메뉴**: Shift+B (가방), Shift+I (내 정보)

## 📱 플레이 방법 (Mobile)
- **이동**: 가상 조이스틱
- **전투**: 스킬 버튼 터치
- **전체화면**: 하단 📺 버튼 클릭

## 🔄 최근 업데이트 내역 (v0.2)
### 1. 전투 시스템 및 스킬 리워크
- **기본 공격**: 관통형 레이저 공격으로 변경 (마나 1 회복)
- **매직 미사일**: 호밍 유도 기능 (마나 2 소모)
- **파이어볼**: 직선 범위 공격 및 화상 도트 데미지 추가 (마나 5 소모)
- **보호막**: 3분간 피해 80% 감소 및 마나 우선 소모 (마나 20 소모)

### 2. 고도화된 몬스터 AI 및 전리품
- **추격 AI**: 몬스터가 일정 거리 내의 플레이어를 감지하고 추격
- **공격 AI**: 플레이어 근접 시 주기적인 데미지 발생
- **회복 시스템**: 몬스터 처치 시 확률적으로 HP 회복 구슬(+) 드랍
- **무한 스폰**: 맵 전체 영역에 대해 3초마다 몬스터 자동 생성

### 3. 기동성 강화: 달리기 시스템
- **가속**: 1초 이상 연속 이동 시 '달리기' 상태 진입 (속도 1.3배)
- **유예 시간**: 방향 전환 시 0.5초의 유예 시간을 주어 달리기 상태가 끊기지 않도록 개선
- **이펙트**: 달리기 시 발자국 먼지 파티클 효과 추가

### 4. 편의성 및 모바일 최적화
- **미니맵 추가**: 우측 상단에 플레이어(백색) 및 몬스터(적색) 위치 실시간 표시
- **S23 최적화**: 모바일 화면 크기에 맞게 UI 규모를 1.5배 축소하고 엄지손가락 영역에 최적화된 배치
- **전체화면 지원**: 브라우저 UI 없이 플레이 가능한 Fullscreen API 연동

## 🛠 기술 스택
- **Language**: JavaScript (Vanilla JS ES6+)
- **Graphics**: HTML5 Canvas API
- **Styling**: Vanilla CSS
- **Assets**: Midjourney / DALL-E 기반 픽셀 아트 소스
