# 🎮 Yurika Online (유리카 온라인)

**Yurika Online**은 HTML5 Canvas와 Firebase를 기반으로 구축된 최첨단 에이전트 기반 MMORPG입니다. 이 프로젝트는 현대적인 웹 기술을 사용하여 부드러운 멀티플레이어 경험과 화려한 마법 전투 시스템을 구현하는 것을 목표로 합니다.

---

## 🚀 주요 기능

- **실시간 멀티플레이어**: Firebase Realtime Database를 통한 실시간 위치 동기화 및 전투 시스템.
- **다이내믹 전투 시스템**: 체인 라이트닝, 매직 미사일, 파이어볼 등 화려한 이펙트와 함께하는 마법 시퀀스.
- **스마트 몬스터 AI**: 플레이어를 감지하고 추적하며, 그룹으로 협력하여 공격하는 지능형 AI 패턴.
- **퀘스트 & 보상**: 슬라임 처치 퀘스트와 대왕 슬라임 레이드를 통한 캐릭터 성장 시스템.
- **사용자 커스터마이징**: 구글 로그인을 통한 간편한 세션 유지 및 캐릭터 이름/스탯 커스터마이징.

---

## 🛠️ 기술 스택

- **Frontend**: Vanilla JavaScript (ES6+), HTML5 Canvas, CSS3
- **Backend/DB**: Firebase Authentication, Firebase Realtime Database
- **Build/Dev**: Live-server (PWA 지원)

---

## 📋 최신 업데이트 내역 (v0.00.03)

### 버그 수정 및 안정화
- **몬스터 사망 로직 개선**: 몬스터가 체력이 0이 되었을 때 사라지지 않던 좀비 버그를 해결했습니다. 이제 모든 클라이언트에서 즉시 사망 연출(Fade-out)과 보상 드랍이 동기화됩니다.
- **코드 안정성 확보**: `Monster.js`에서 발생하던 문법 오류를 수정하여 게임 로딩이 멈추는 현상을 해결했습니다.

### 시스템 개선
- **구글 로그인 최적화**: 세션 유지(Persistence)를 로컬로 명시하여 새로고침 시에도 로그인이 유지되도록 개선했습니다. 캐릭터 생성 시 구글 닉네임을 자동으로 불러옵니다.
- **퀘스트 시스템 고도화**: 몬스터 처치 즉시 퀘스트 카운트가 반영되도록 수정하였으며, 보상 수령 시 시각적인 효과를 강화했습니다.
- **슬라임 AI 강화**: 레벨 1 플레이어에게도 공격적으로 반응하도록 인식 범위를 조정하여 초반 긴장감을 높였습니다.

### 개발 환경 설정
- **호스트 권한 동기화**: 멀티플레이어 시 환경 안정성을 위해 호스트 권한 동기화 및 유령 사용자(Ghost Users) 정리 로직을 강화했습니다.

## 🕹️ 시작하기

1. 저장소를 클론합니다.
2. `npm start` 명령어로 로컬 서버를 실행합니다.
3. `http://localhost:8081`에 접속하여 게임을 즐기세요!
