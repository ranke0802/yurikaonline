# 🎮 Yurika Online (유리카 온라인) - v0.24.4

[**🎮 Firebase 라이브 데모**](https://yurika-online.web.app/)

## 📜 최신 업데이트 (v0.24.4) - 2026-01-25
- **원본 소스(`yurikaonline-master`) 기반 화질 고도화**:
  - 원본 소스의 리사이즈 및 줌(Zoom) 밸런스를 이식하여 모바일에서 가장 보기 편안한 가시 거리를 확보했습니다.
  - 고해상도 디스플레이에서 발생하는 과도한 픽셀 깨짐을 방지하기 위해 브라우저 최적 렌더링(Anti-aliasing) 방식을 채택, 훨씬 부드럽고 선명한 고품질 화면을 구현했습니다.
- **v0.24.3 포함**: HiDPI 대응, 물리 해상도 확장.
- **v0.24.1 핫픽스 포함**: 초기 로딩 렌더링 글리치 및 카메라 참조 오류 수정 완료.
- **로컬 개발 환경 지원**: `npm start`를 통한 로컬 개발 서버 및 핫 리로딩 환경이 구축되었습니다. ([설치 가이드](LOCAL_DEV.md))
- **마법진 이펙트 확장**: 캐릭터 하단 마법진의 크기를 1.5배 키워 시각적 박진감을 높였습니다.
- **스탯 기반 쿨타임 감소(CDR)**: 지력(INT)과 지혜(WIS)를 올릴수록 스킬 재사용 대기시간이 줄어집니다. (최대 75% 감소)
- **밸런스 및 안정성 개선**:
  - 슬라임 이동 속도를 20px로 재조정하여 초반 쾌적함을 개선했습니다.
  - 일반 공격(J)과 매직 미사일(H)의 쿨타임 및 사거리를 재조정했습니다.
  - 멀티플레이어 환경에서 다른 플레이어 위치 보간 관련 런타임 오류를 수정했습니다.

## 🛠️ 로컬 개발 환경 구축
1. **의존성 설치**: `npm install`
2. **실행**: `npm start` (브라우저에서 `localhost:8080` 접속)

## 🎮 조작 방법
- **이동**: 화면 좌측 조이스틱 또는 방향키
- **기본 공격 (J)**: 체인 라이트닝 (연쇄 번개)
- **스킬 (H)**: 매직 미사일 (유도 레이저)
- **스킬 (U)**: 파이어볼 (광역 폭발)
- **스킬 (K)**: 앱솔루트 베리어 (방어막)

## 🚀 주요 기능
- **실시간 멀티플레이어**: 다른 플레이어와 같은 필드에서 모험을 즐길 수 있습니다.
- **성장 시스템**: 몬스터를 처치하고 경험치를 얻어 레벨업하고 스탯/스킬을 강화하세요.
- **자동 저장**: 플레이어의 데이터가 Firebase를 통해 실시간으로 안전하게 저장됩니다.
